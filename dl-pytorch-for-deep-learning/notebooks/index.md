## Running Notebooks Locally

To run these notebooks locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/rajdeepd/dl-pytorch-cert.git
   cd dl-pytorch-cert
   ```

2. Install dependencies:
   ```bash
   pip install torch torchvision numpy matplotlib jupyter
   ```

3. Launch Jupyter:
   ```bash
   jupyter notebook
   ```

## Running on Google Colab

You can also open any notebook directly in Google Colab by:
1. Navigating to the notebook on GitHub
2. Changing `github.com` to `githubtocolab.com` in the URL
